# Animate UI Components

The animated icon wrappers, icon animations, and related animation primitives
in this directory are adapted from [Animate UI](https://animate-ui.com/) by
Elliot Sutton (`imskyleen/animate-ui`). The source was matched against commit
`efeb96ffd7a3b7a4868667e4ac3c346620fb3044`.

Animate UI is used here as part of the Sylva Signer application. Its MIT plus
Commons Clause license permits application use and distribution while
restricting sale or redistribution of the components themselves in their
original form. The complete license is preserved at
`licenses/animate-ui.txt`.

The underlying icon designs are based on Lucide. Lucide's ISC license and the
MIT notice for Feather-derived icons are preserved at
`licenses/npm/lucide-react.txt`.
